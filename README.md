## LixSketch - The Canvas You Can Trust 🖌️
### LixSketch is an open source freemium platfor for collaborative canvas for technical presentation / modelling on web + IDE + pwa (under development)

<img width="1322" height="612" alt="og-image" src="https://github.com/user-attachments/assets/a96e85be-dce9-464c-8451-5513ad521dd6" />

### Understanding why you need LixSketch

LixSketch offers technical / professional proficiency in quick canvases used for any technical interview / presentation / workshop / classroom in a collaborative fashion, which can be shared with anyone at any time in a matter of a click. 

<img width="1867" height="874" alt="image" src="https://github.com/user-attachments/assets/9971cf94-dba8-4436-b858-4d80e99b246d" />

### Why LixSearch stands out 

- You don't need to login or create an account to use.
- Has a generous freemium model.
- We have our own scripting language [LixScript](https://sketch.elixpo.com/docs)
- Integrates Excalidraw, TLDraw, Eraser io inside one single workspace.
- It's Open Source with MIT OSI License
- Built in Public understanding the needs of the developers and users
- Quick and Free (Supports Generous Guest Mode Functions)
> Explore us to know more at [LixSketch](https://sketch.elixpo.com)
